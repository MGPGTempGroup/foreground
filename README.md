# MGPG

# 环境、依赖
使用Vue官方脚手架：`Vue-cli 3`

使用`vue-router`构建单页应用

使用`ElementUI`作为第三方组件库，以及栅格布局（响应式）

# 项目运行方式
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
